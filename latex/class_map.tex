\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename K, typename V$>$\newline
class Map$<$ K, V $>$}

A template wrapper class for the standard C++ std\+::map container. 

This class provides a dynamic, heap-\/allocated wrapper around std\+::map, implementing the Rule of Three (Destructor, Copy Constructor, Assignment Operator) and providing specific methods for insertion and retrieval that handle key and value pointers, as required by the application\textquotesingle{}s design.
