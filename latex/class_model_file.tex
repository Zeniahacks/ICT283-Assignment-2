\doxysubsection{Detailed Description}
Manages all models and room templates. 

Models are constructed using Harry\textquotesingle{}s block class. Room templates contain references to several models, and are used to place furniture in rooms. Models do not support collision detection.

\mbox{\hyperlink{struct_model_file_1_1_template}{Template}} names can contain multiple words (ie with spaces).~\newline
Case differences are ignored in template names, but not model names.~\newline
Models are standard bdl files.~\newline
