\doxysubsection{Detailed Description}
The main application class for Assignment 2, managing the user interface and data processing flow. 

This class handles the main menu loop, user input, and delegates data-\/related tasks to the {\ttfamily \mbox{\hyperlink{class_weather_data_collection}{Weather\+Data\+Collection}}} object. It ensures data is loaded before analysis options are offered.
