\doxysubsection{Detailed Description}
Represents a specific point in time (day, month, year, hour, minute). 

This class stores date and time components and provides utility methods for access, modification, string conversion, and comparison, specifically designed to work with pointer comparisons needed by the \mbox{\hyperlink{class_bst}{Bst}} class.
